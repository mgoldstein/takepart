
-- SUMMARY --

Terms TagIt allows users to use tagging format for term reference
autocomplete fields. This enhances the overall user experience
with usage.

-- PREREQUISITE --

* Download Tags Input https://github.com/xoxco/jQuery-Tags-Input
* This goes into your /sites/all/libraries folder

-- CONFIGURATION --

1. Add a new field in your node with:
  - Field type: term reference
  - Form element: Autocomplete term widget (tagging)
2. In edit scroll to the TagIt field group.
3. Check off "Enable TagIt on this field" to use tagit
4. Check off "Enable autocomplete" to have a drop down of terms
